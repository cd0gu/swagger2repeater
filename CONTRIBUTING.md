# Contributing to Swagger2Repeater

Thank you for considering contributing!

## How to Contribute

### 1. Fork the repository  
Click **Fork** at the top-right of the GitHub page.

### 2. Create a feature branch  

git checkout -b feature/my-improvement

### 3. Follow code guidelines
- The project must run on **Jython**, not CPython  
- Use Python 2.7â€“compatible syntax  
- Avoid external libraries unless bundled with Burp/Jython

### 4. Test in Burp Suite
Ensure the extension loads without warnings.

### 5. Submit a Pull Request
Include:
- Description of the change  
- Before/after behavior (if UI-related)  
- Sample Swagger files if needed  

Thank you for contributing!
